# Padronização de cache/versionamento

Substitua os arquivos deste pacote na raiz do projeto.

Correção aplicada:
- Todos os CSS locais em todas as páginas HTML agora usam a mesma versão:
  ?v=20260703-7
- Todos os JS locais em todas as páginas HTML agora usam a mesma versão:
  ?v=20260703-7
